
# Basics

This repository contains a collection of Ruby scripts for basic number and string operations. Each script is designed to perform a specific task, such as checking for anagrams, generating Fibonacci numbers, validating Armstrong numbers, and more.

## Features

- Anagram Checker
- Armstrong Number Generator & Validator
- Even/Odd Checker
- Factorial Calculator
- Fibonacci Generator
- GCD Finder
- LCM Calculator
- Leap Year Checker
- Number Reversal
- Palindrome Checker
- Prime Checker
- Prime Factorization
- Sum of Digits
- Vowel/Consonant Counter

## Usage

To run any script, use the following command:

```bash
ruby <script_name>.rb
```

Replace `<script_name>` with the desired file, e.g., `factorial_calculator.rb`.

## File Naming

Each feature is available in both snake_case and CamelCase file formats for convenience.
